2. Tipologia di dati trattati

2.1 Dati salvati esclusivamente sul dispositivo

Haven salva sul dispositivo dell’utente:
	•	Contatti fidati (nome, numero opzionale, preferenze)
	•	Check-in e stato sessioni “Signal”
	•	Safe zones (nome, categoria, coordinate geografiche)
	•	Contenuti del Vault (note, indirizzi, informazioni mediche, dati inseriti manualmente)
	•	Log attività interni
	•	Preferenze tecniche

I contenuti del Vault sono cifrati localmente.

Questi dati non vengono trasmessi ai server Haven.

⸻

2.2 Dati trasmessi al backend (Supabase)

Per abilitare funzionalità di collegamento tra utenti:
	•	Identificatore pseudonimo anonimo
	•	Trusted links (ID utenti, stato invito, timestamp)
	•	Trusted inbox (alert, stato sessione, scadenza, metadata)

Non vengono trasmessi:
	•	numeri di telefono
	•	email come identificatori primari
	•	posizione precisa dell’utente
	•	contenuti del Vault

⸻

2.3 Dati di utilizzo (solo con consenso esplicito)

Se l’utente fornisce consenso:
	•	Timestamp evento
	•	Fascia oraria
	•	Giorno settimana
	•	City ID approssimato
	•	Versione app e piattaforma
	•	Eventi di prodotto (es. session_start, session_end)

Non vengono raccolti:
	•	coordinate precise
	•	messaggi
	•	audio
	•	testo libero

Retention raw analytics: massimo 14 giorni.

⸻

2.4 Servizi Apple / iOS

Haven utilizza servizi Apple quali:
	•	MapKit per ricerca luoghi e percorsi
	•	Notifiche locali
	•	Apple Watch snapshot minimo
	•	Sistemi biometrici iOS (senza memorizzazione biometrica)

⸻

3. Finalità del trattamento

I dati sono trattati per:
	•	Fornire funzionalità di sicurezza personale
	•	Gestire sessioni di monitoraggio
	•	Consentire collegamenti tra utenti fidati
	•	Migliorare il prodotto (solo previo consenso)

⸻

4. Base giuridica
	•	Esecuzione del servizio richiesto
	•	Consenso esplicito (analytics)
	•	Legittimo interesse alla sicurezza del servizio

⸻

5. Conservazione
	•	Dati locali: fino alla disinstallazione
	•	Dati backend: fino alla cancellazione dell’utente
	•	Analytics: massimo 14 giorni (raw)

⸻

6. Diritti dell’utente

L’utente può richiedere:
	•	Accesso ai dati
	•	Rettifica
	•	Cancellazione
	•	Revoca consenso analytics

Richieste via email al titolare.

⸻

7. Sicurezza

Haven adotta:
	•	Minimizzazione dei dati
	•	Identificatori pseudonimi
	•	Cifratura locale del Vault
	•	Nessun tracciamento pubblicitario
